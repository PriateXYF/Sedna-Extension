# Sedna-Extension

基于 Leancloud 的自动化签到工具。

* Sedna
* [说明文档](https://sedna.virts.app/)

## 安装方式

```bash
git clone 'https://github.com/PriateXYF/Sedna-Extension.git'
cd Sedna-Extension
yarn
yarn build
mv dist Sedna-Extension
```

打开 chrome://extensions ，开启 `开发者模式` ，选择 `加载已解压的拓展程序` ，选择 `Sedna-Extension` 。
